$ wget https://raw.githubusercontent.com/calliapp/cli/main/calli

The CLI is a Bash script with a Python shebang, so you need Python3 installed, as well as the modules found in requirements.txt

Your token is stored in .token.json. Do not share this. It will give anyone access to your account.

MacOS Install

To be able to run $ calli from everywhere, and not just ./calli from the directory, move calli to /usr/local/bin before logging in

$ git clone https://github.com/calliapp/cli

$ cd cli

$ python3 -m pip install -r requirements.txt

$ mv calli /usr/bin/local
