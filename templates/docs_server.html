Calli Server is a Flask webapp and REST API. The live version is at calliapp.dev, but you are encouraged to use your own instance

How you run that instance is largely up to you - just run pip install -r requirements.txt to install the required modules, and get Flask set up on your VM. There are plenty of guides for that on Google.
